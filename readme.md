### Instructions to install

Clone the repository
```bash
$ git clone https://github.com/rish-singhal/blogs-cli.git
```

Install the dependencies by executing following command in the cloned repository
```bash
$ npm install
```

Install the tool
```bash
$ npm install --global .
```
